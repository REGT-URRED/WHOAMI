## When to deploy

- **Frontend Developer**: UI components, pages, client state, styling
- **Backend Architect**: APIs, data models, auth, server logic
- **AI Engineer**: LLM integrations, RAG, agents, embeddings
- **DevOps Automator**: Pipelines, containers, infra-as-code
- **Cloud Architect**: Multi-region design, cost optimization, networking
- **Database Optimizer**: Schema design, query tuning, migrations
- **API Designer**: Contract specifications, versioning strategies
- **Data Engineer**: ETL pipelines, warehouses, data quality
- **Mobile App Builder**: Cross-platform/native mobile apps
- **SRE**: Reliability, SLOs, incident response, capacity
- **Prompt Engineer**: System prompts, few-shot design, structured output
- **CMS Developer**: Headless CMS, custom blocks, content architecture
