# Security Division — 🔒

Defense in depth. From code audits to incident response.

| Agent | Emoji | Tier | Tools |
|-------|-------|------|-------|
| Security Auditor | 🔒 | 1 | burp-suite, semgrep, trivy, snyk, zap |
| Penetration Tester | 🛡️ | 2 | metasploit, burp-suite, nmap, wireshark, hydra |
| Incident Responder | 🚨 | 2 | splunk, elastic, thehive, velociraptor, osquery |

## When to deploy

- **Security Auditor**: OWASP audits, secrets scanning, auth review, dependency checks
- **Penetration Tester**: Simulated attacks, exploitation reports, infrastructure testing
- **Incident Responder**: Active breaches, containment, forensics, postmortems
