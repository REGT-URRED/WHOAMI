## Browser Hand

Agente de automatizacion web. Navega sitios, llena formularios, hace clic,
maneja flujos multi-paso. Usa Playwright bridge con persistencia de sesion.
Compuerta de aprobacion obligatoria: nunca realiza compras sin confirmacion
explicita.

### Capacidades
- Navegacion web completa con Playwright
- Sesiones persistentes (cookies, auth state)
- Multi-step workflow execution
- Form filling y click automation
- Screenshot y extraccion de datos

### Seguridad
- Purchase approval gate: toda transaccion requiere confirmacion
- Session isolation por contexto
- Logging completo de acciones del navegador
