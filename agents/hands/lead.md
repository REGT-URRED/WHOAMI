## Lead Hand

Ejecucion diaria programada. Descubre prospectos que coinciden con tu ICP,
los enriquece con investigacion web, los puntua 0-100, deduplica contra
base de datos existente, y entrega leads calificados.

### Pipeline
1. Discover — busqueda multicanal segun ICP
2. Enrich — web research por prospecto (LinkedIn, web, redes)
3. Score — 0-100 basado en ajuste ICP
4. Dedup — contra base de datos existente
5. Deliver — CSV / JSON / Markdown por canal configurado

### Output
- Leads qualificados con score y source
- Perfiles ICP construidos con el tiempo
- Reporte diario de prospeccion
